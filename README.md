## ⚡ The Physics Behind the Invention

<div align="center">

```
Human memory doesn't store things like a hard drive.
It stores them like a wave function.

     probability
          │
     1.0  │   *
          │  * *                    ← vivid memory (high salience)
     0.5  │ *   *   *
          │*     * * *   *   *      ← fading (temporal decay)
     0.0  └──────────────────────► time

  M(t) = M₀ · e^(-t/S)            ← Ebbinghaus forgetting curve
  P(ψ) = |⟨ψ_recall|ψ_stored⟩|²   ← retrieval as wave collapse

The Temporal Memory™ engine models exactly this.
Memory exists in superposition until the moment of recall.
Recall collapses the wave. That's not a metaphor. That's the architecture.
```
