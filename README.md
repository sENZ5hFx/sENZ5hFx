## ⚡ The Physics Behind the Invention

<div align="center">

```
Human memory doesn't store things like a hard drive.
It stores them like a wave function.

     probability
          │
     1.0  │   *
          │  * *                    ← vivid memory
     0.5  │ *   *   *
          │*     * * *   *   *      ← fading
     0.0  └─────────────────────► time

The Temporal Memory™ engine models this exactly.
Ebbinghaus decay curve + quantum superposition metaphor:
memory exists in all states until you recall it.
The act of recall collapses it into certainty.

    M(t) = M₀ · e^(-t/S)   ← Ebbinghaus forgetting curve
    P(ψ) = |⟨ψ_recall|ψ_stored⟩|²  ← retrieval as wave collapse
```
